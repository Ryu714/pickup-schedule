# 등하교/등하원 시간표

두 자녀(최윤우, 류준)의 일일 등하교·등하원 시간표를 한눈에 확인할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- **함께보기** — 두 자녀의 일정을 나란히 비교
- **개별 시간표** — 자녀별 상세 일정 확인
- **요일별 조회** — 월~금 요일 선택, 오늘 요일 자동 선택
- **현재 시간 인디케이터** — 함께보기에서 현재 시간을 빨간 선으로 표시
- **학원별 색상 구분** — 영어, 피아노, 과학, 축구 등 활동별 색상 코드

## 기술 스택

HTML5 · CSS3 · Vanilla JavaScript (단일 `index.html` 파일)

## 실행 방법

`index.html`을 브라우저에서 열면 바로 사용할 수 있습니다.
