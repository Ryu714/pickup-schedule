# 등하교/등하원 시간표

두 자녀(최윤우, 류준)의 일일 등하교·등하원 시간표를 한눈에 확인할 수 있는 모바일 웹 애플리케이션입니다.

## 주요 기능

- **함께보기** — 두 자녀의 일정을 나란히 비교 (사이드 바이 사이드 레이아웃)
- **요일별 조회** — 월~금 요일 탭, 오늘 요일 자동 선택
- **현재 시간 인디케이터** — 빨간 선 + 펄스 도트로 현재 시각 표시
- **다음 일정 강조** — 다가올 가장 가까운 카드에 glow 애니메이션
- **지난 일정 디밍** — 이미 지난 시간의 카드 자동 흐리게 처리
- **다크 모드** — shadcn/zinc 스타일, 해/달 토글로 수동 전환 (첫 방문 시 라이트 기본, localStorage 유지)
- **날씨 위젯** — Open-Meteo API 기반 현재 기온 + 최고/최저 표시
- **학원별 보더 색상** — 같은 학원의 등원-하원 카드를 보더 색상으로 그룹핑
- **반응형** — 모바일 최적화 (420px, 360px 브레이크포인트)

## 기술 스택

HTML5 · CSS3 · Vanilla JavaScript (단일 `index.html` 파일)

## 실행 방법

`index.html`을 브라우저에서 열면 바로 사용할 수 있습니다.
